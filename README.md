meetphp-8

Projects and Presentation for MeetPHP#8


Apps
===========

Silex & Laravel4 - simple skeleton app with a controller and Doctrine display from memcache/db 10k records.


Installation:

1. [http://getcomposer.org/]

2. Depends on App:
  2.1 Silex:
    
    $ cd silex
    $ composer install


  2.2 Laravel:
   
   $ cd lavarel4
   $ composer install
